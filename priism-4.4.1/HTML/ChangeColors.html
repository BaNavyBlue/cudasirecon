Changing Graphic Colors

Overview

This application allows you to change the colors that are used for overlay graphics on the image window.

The RGB values and names for the overlay colors are displayed in a list and can be edited by hand. You can also select a color from the color wheel. To do this, depress the toggle button for the graphic color you want to edit. Then click in the color wheel to select the color; the selected color will be displayed as a bar below the wheel. To adjust the brightness of the color, slide and release the shade slider. When you're satisfied with the color, press the apply color button to update the RGB values in the list of overlay colors.

Depressing the Accept Colors button causes the current colors to be applied. If you want to use the same set of colors in later Priism sessions, use the Save Colors to File to save the settings which can then be restored with Retrieve Colors from File. If you follow a simple convention, you will be able to select your edited colors from the Priism preferences dialog.

Filenames

The Save Colors to File brings up a dialog in which you enter the filename for the colors file. If a path is not specified, the file is written to the directory from which Priism was started.

To make your colors file available in the Priism preferences dialog, the file needs to have the .colors extension and be located in the .iveprefs directory in your home directory (if this directory does not exist, you'll need to create it).
