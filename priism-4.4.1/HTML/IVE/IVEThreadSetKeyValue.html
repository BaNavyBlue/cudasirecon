Reference: IVEThreadSetKeyValue

Overview

IVEThreadSetKeyValue associates a thread-specific value with a key. Different threads may associate different values with the same key.

Prototype (C)

#include "ive_standards.h"
#include "ive_thread.h"
void IVEThreadSetKeyValue(void* value, IVEThreadKey key, int* p_status);

Prototype (Fortran)

There is no Fortran interface.

Parameters

value
This is the value to associate with key
key
key is a handle to thread-local storage initialized with IVEThreadInitKey. The results are not define when IVEThreadSetKeyValue is called with an uninitialized key or a key that has been deleted.
p_status
If the call fails, *p_status will be set to one of the values below; otherwise its value is not modified.
IVE_ERR_NO_MEM
There was insufficient memory available to complete the operation.
IVE_ERR_INVALID
key is invalid.
IVE_ERR_UNRECOGNIZED
The operating system or system library returned an error code that was not expected.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.

Cross references

IVEThreadInitKey, IVEThreadDestroyKey, IVEThreadGetKeyValue, and IVEThreadKey
