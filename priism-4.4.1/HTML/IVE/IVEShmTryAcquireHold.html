Reference: IVEShmTryAcquireHold

Overview

Locks a hold if it is currently unlocked.

Prototype (C)

#include "ive_standards.h"
#include "ive_shm.h"
int IVEShmTryAcquireHold(IVEShmHold* p_hold);

Prototype (Fortran)

There is no Fortran interface.

Parameters

p_hold
p_hold points to the hold to be locked.

Return value

If *p_hold could be locked, the function returns true and the caller should release the lock with IVEShmReleaseHold when it no longer needs to lock the hold. If *p_hold could not be locked because it was already locked, the function returns false.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.

Cross references
