Reference: IVEShmGetSemaphoreValue

Overview

Returns a snapshot of a semaphore's value. It is only a snapshot because subsequent actions by other threads may modify the value.

Prototype (C)

#include "ive_standards.h"
#include "ive_shm.h"
int IVEShmGetSemaphoreValue(IVEShmSemaphore* p_sema);

Prototype (Fortran)

There is no Fortran interface.

Parameters

p_sema
p_sema points to the semaphore whose value is returned.

Return value

Returns the value of the semaphore at some point in time between the start of the call and when the call returns.

Bugs

On Mac OS X, IVEShmGetSemaphoreValue always returns zero because the system call to retrieve the semaphore value is not implemented. To check for this bug at compile time, you can check if the macro IVECONF_UNIMPL_SEM_GETVALUE is defined.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
