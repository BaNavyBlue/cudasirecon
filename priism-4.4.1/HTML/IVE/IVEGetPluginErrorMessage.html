Reference: IVEGetPluginErrorMessage

Overview

Returns an explanation for the last error encountered by IVELoadPlugin since a previous call to IVEGetPluginErrorMessage.

Prototype (C)

#include "ive_plugin.h"
const char* IVEGetPluginErrorMessage(void);

Prototype (Fortran)

There is no Fortran interface.

Return value

Either returns NULL or a pointer to a null-terminated string describing the last error encountered by IVELoadPlugin. The pointer returned by IVEGetPluginErrorMessage may refer to a static buffer whose contents could be overwritten by subsequent calls to IVELoadPlugin.
