Reference: IVEShmInitSemaphore

Overview

Initializes a semaphore usable by threads in different processes.

Prototype (C)

#include "ive_standards.h"
#include "ive_shm.h"
void IVEShmInitSemaphore(unsigned int initial_value, IVEShmSemaphore* p_sema, int* p_status);

Prototype (Fortran)

There is no Fortran interface.

Parameters

initial_value
When the semaphore is initialized, its value is initial_value.
p_sema
p_sema points to the semaphore to be initialized. When the semaphore is no longer needed, any resources associated with it may be freed by calling IVEShmDestroySemaphore.
p_status
If the call fails, *p_status will be set to one of the values below; otherwise its value is not modified.
IVE_ERR_NO_RESOURCES
There was insufficient system resources available to complete the operation.
IVE_ERR_NO_PERMISSION
The caller does not have sufficient privileges to perform the operation.
IVE_ERR_EXIST
p_sema points to a semaphore that has been previously initialized and not subsequently destroyed.
IVE_ERR_UNSUPPORTED
The system does not support the type of semaphores that the library expected to use.
IVE_ERR_INVALID
initial_value exceeds the maximum value that may be assigned to a semaphore.
IVE_ERR_INTERNAL
An error internal to the library was detected. This is most likely because of a bug in the library.
IVE_ERR_UNRECOGNIZED
The operating system or system library returned an error code that was not expected.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
