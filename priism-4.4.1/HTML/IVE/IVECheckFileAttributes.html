Reference: IVECheckFileAttributes

Overview

Determines whether a given file name exists, is a directory, and is accessible to the caller.

Prototype (C)

#include "ive_file.h"
int IVECheckFileAttributes(const char* name, int mask);

Prototype (Fortran)

integer function IVECheckFileAttributes(name, mask)
character*(*) name
integer mask

Parameters

name
(in) name is the name of the file to check. For Fortran callers, any trailing spaces are not considered part of the filename.
mask
(in) mask defines which attributes to check. It may be a bitwise-or of the constants listed below. In Fortran you can use an additive combination; the constants are defined in ive_file.inc.
IVE_FILE_CHECK_EXIST
Verify whether or not a file named name exists.
IVE_FILE_CHECK_DIR
Verify whether or not the file named name is a directory.
IVE_FILE_CHECK_ACCESS
Verify whether or not the file named name is readable, writable, or executable by the caller.
IVE_FILE_CHECK_ALL
Is a shortcut for checking all of the above attributes.

Return value

The return value is a bitwise-or of the constants defined below. For Fortran, the constants are defined in ive_file.inc. The return value reflects a snapshot of the file's status; it is possible that the file's attributes may change between the completion of IVECheckFileAttributes and when the caller later attempts to operate on the file.

IVE_FILE_INDETERM
The path to name contains at least one directory for which the caller does not have search privileges. So it is not possible to determine if the file exists and what its attributes are. If IVE_FILE_INDETERM is included in the return value, no other value will be. IVE_FILE_INDETERM is only set if IVE_FILE_CHECK_EXIST is included in mask.
IVE_FILE_EXIST
A file named name exists. IVE_FILE_EXIST is only set if IVE_FILE_CHECK_EXIST is included in mask
IVE_FILE_DIR
The file named name exists and is a directory. IVE_FILE_DIR is only set if IVE_FILE_CHECK_DIR is included in mask.
IVE_FILE_READ
The file named name exists and may be read by the caller. IVE_FILE_READ is only set if IVE_FILE_CHECK_ACCESS is included in mask.
IVE_FILE_WRITE
The file named name exists and may be written to by the caller. IVE_FILE_WRITE is only set if IVE_FILE_CHECK_ACCESS is included in mask.
IVE_FILE_EXEC
The file named name exists and may be executed by the caller. IVE_FILE_EXEC is only set if IVE_FILE_CHECK_ACCESS is included in mask.
