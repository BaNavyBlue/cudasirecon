Reference: IVERemoveInput

Overview

Removes an input/output event source registered with IVEAddInput.

Prototype (C)

#include "ive_eventloop.h"
void IVERemoveInput(IVEInputId id, IVEEventContext context);

Prototype (Fortran)

subroutine IVERemoveInput(id, context)
integer id, context

Parameters

id
id is a value returned by a previous call IVEAddInput.
context
context is the context to which id belongs.

Cross references

IVEAddInput, IVEEventContext, and IVEInputId
