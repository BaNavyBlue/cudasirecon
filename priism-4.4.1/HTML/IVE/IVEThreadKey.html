Reference: IVEThreadKey

Overview

An IVEThreadKey is a handle to thread-specific storage. Once a key has been initialized with IVEThreadInitKey, a thread can use IVEThreadSetKeyValue, and IVEThreadGetKeyValue to store or retrieve the value of a pointer. The value of the stored pointer is local to the calling thread (calls to IVEThreadSetKeyValue and IVEThreadGetKeyValue from different threads using the same key are completely independent).

Include ive_standards.h and ive_thread.h to declare IVEThreadKey. Beware that inclusion of ive_standards.h sets feature-set macros. It is safest to include it before any other header.
