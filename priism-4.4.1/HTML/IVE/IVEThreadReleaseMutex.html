Reference: IVEThreadReleaseMutex

Overview

Unlocks a mutual exclusion lock.

Prototype (C)

#include "ive_standards.h"
#include "ive_thread.h"
void IVEThreadReleaseMutex(IVEThreadMutex* p_mutex);

Prototype (Fortran)

There is no Fortran interface.

Parameters

p_mutex
p_mutex points to the mutex to be unlocked. Only the thread which locked the mutex should unlock it.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
