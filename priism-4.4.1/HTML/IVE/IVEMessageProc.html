Reference: IVEMessageProc

Overview

In C, IVEMessageProc is the prototype for callback routines registered with IVESetMessageHandler. As shown below, it takes three arguments: the argument registered when the callback was registered, the identifier for the message type, and the event context. The Fortran interface is similar and is also shown below.

Include ive_eventloop.h to declare IVEMessageProc.

Prototype (C)

void (*IVEMessageProc)(void* arg, IVEMessage msg, IVEEventContext context);

Prototype (Fortran)

subroutine your_name_here(arg, msg, context)

integer arg(*)

integer msg, context

Parameters

arg
arg is the argument that was registered with IVESetMessageHandler when the callback routine was registered.
msg
msg is the integer identifying which message was received.
context
context is the event context in which the event occurred.

Cross references

IVESetMessageHandler, IVECancelMessageHandler, IVEEventContext, and IVEMessage
