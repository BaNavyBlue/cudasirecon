Reference: IVEGetWarningHandler

Overview

Returns the address of the function call when IVEReportWarning is called.

Prototype (C)

#include "ive_processbase.h"
IVEWarningHandler IVEGetWarningHandler(void);

Prototype (Fortran)

There is no Fortran interface.

Return value

The current handler is returned.
