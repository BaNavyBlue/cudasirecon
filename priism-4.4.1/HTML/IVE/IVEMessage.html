Reference: IVEMessage

Overview

An IVEMessage is an integer used to identify a class of messages for use with IVESetMessageHandler and IVEPostMessage. Only values between IVE_MESSAGE_MIN and IVE_MESSAGE_MAX are recognized as valid messages. Those limits are defined in ive_eventloop.h for C and ive_eventloop.inc for Fortran. IVE_MESSAGE_MIN - IVE_MESSAGE_MAX + 1 is guaranteed to be at least 255.
