Reference: IVESetArenaAttrSize

Overview

Changes the arena size (the maximum amount of space the arena can occupy) in a set of arena attributes. The default size if one has not been supplied is 300 Megabytes. The space overhead for the arena is on the a handful of Kilobytes.

Prototype (C)

#include "ive_shm.h"
void IVESetArenaAttrSize(IVEArenaSize sz, IVEArenaAttr attr);

Prototype (Fortran)

There is no Fortran interface.

Parameters

sz
sz is the desired size in bytes.
attr
attr is the attributes object to modify.
