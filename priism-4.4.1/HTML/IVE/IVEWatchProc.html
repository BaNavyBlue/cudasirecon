Reference: IVEWatchProc

Overview

In C, IVEWatchProc is the prototype for the callback routine registered with IVESetWatchProc. The routine is called when there is a change to the set of file descriptors (handles) monitored by the event loop routines.

Include ive_eventloop.h to declare IVEWatchProc.

Prototype (C)

void (*IVEWatchProc)(IVEDesc desc, int reason, void* arg);

Parameters

desc
desc is the file descriptor (handle) that is affected by the change.
reason
reason describes the nature of the change. On POSIX systems, it is a bitwise or of one or more of the following:
IVE_WATCH_ADD_READ
desc is to be monitored for when it is ready for reading.
IVE_WATCH_ADD_WRITE
desc is to be monitored for when it is ready for writing.
IVE_WATCH_ADD_EXCPT
desc is to be monitored for the presence of exceptional (out-of-band) data.
IVE_WATCH_DEL_READ
desc is no longer to be monitored for when it is ready for reading.
IVE_WATCH_DEL_WRITE
desc is no longer to be monitored for when it is ready for writing.
IVE_WATCH_DEL_EXCPT
desc is no longer to be monitored for when it is ready for exceptional (out-of-band) data.
arg
arg is the argument that was passed to IVESetWatchProc when the callback routine was registered.
