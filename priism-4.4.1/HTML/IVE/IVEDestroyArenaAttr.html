Reference: IVEDestroyArenaAttr

Overview

Destroys an attributes object created by IVECreateArenaAttr.

Prototype (C)

#include "ive_shm.h"
void IVEDestroyArenaAttr(IVEArenaAttr attr);

Prototype (Fortran)

There is no Fortran interface.

Parameters

attr
attr is the attributes object to be destroyed; it may be zero. Using the attributes object after it is destroyed will give undefined results.
