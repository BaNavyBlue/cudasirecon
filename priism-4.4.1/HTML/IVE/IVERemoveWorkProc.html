Reference: IVERemoveWorkProc

Overview

Removes a work procedure registered with IVEAddWorkProc.

Prototype (C)

#include "ive_eventloop.h"
void IVERemoveWorkProc(IVEWorkProcId id, IVEEventContext context);

Prototype (Fortran)

subroutine IVERemoveWorkProc(id, context)
integer id, context

Parameters

id
id is a value returned by a previous call IVEAddWorkProc.
context
context is the context to which id belongs.

Cross references

IVEAddWorkProc, IVEEventContext, and IVEWorkProcId
