Reference: IVESetFatalHandler

Overview

Registers the function to be called when IVEReportFatal is called.

Prototype (C)

#include "ive_processbase.h"
IVEFatalHandler IVESetFatalHandler(IVEFatalHandler handler);

Prototype (Fortran)

There is no Fortran interface.

Parameters

handler
handler is the routine to be registered. The documentation for IVEFatalHandler describes the arguments and behavior that handler is assumed to have. If handler is zero (NULL), the default handler, which prints a message to standard error and causes the program to exit, is restored.

Return value

The previous handler is returned.
