Reference: IVERemoveSignal

Overview

Removes an event source registered with IVEAddSignal.

Prototype (C)

#include "ive_eventloop.h"
void IVERemoveSignal(IVESignalId id, IVEEventContext context);

Prototype (Fortran)

subroutine IVERemoveSignal(id, context)
integer id, context

Parameters

id
id is a value returned by a previous call IVEAddSignal.
context
context is the context to which id belongs.

Cross references

IVEAddSignal, IVEEventContext, and IVESignalId
