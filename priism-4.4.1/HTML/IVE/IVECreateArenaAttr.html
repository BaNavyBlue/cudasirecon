Reference: IVECreateArenaAttr

Overview

Creates an attributes object that is used with IVECreateArena to specify the attributes of a new shared memory segment.

Prototype (C)

#include "ive_shm.h"
IVEArenaAttr IVECreateArenaAttr(void);

Prototype (Fortran)

There is no Fortran interface.

Return value

If the function succeeds, a non-zero value is returned which refers to an attributes object with all settings at their defaults; use IVESetArenaAttrSize and IVESetArenaAttrChunk to modify these settings. If the function call fails, zero is returned. When the attributes are no longer needed, the resources associated with the attributes may be released with IVEDestroyArenaAttr.
