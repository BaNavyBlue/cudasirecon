Reference: IVECreateEventContext

Overview

Creates the context needed by the event loop functions. The context is only safe for access from a single thread of execution at a time. For an event context that can be used with multiple threads, use IVECreateThreadedEventContext.

Prototype (C)

#include "ive_eventloop.h"
IVEEventContext IVECreateEventContext(void);

Prototype (Fortran)

integer function IVECreateEventContext()

Return value

Returns a non-zero value if successful. Returns zero if the context could not be created. When the context is no longer needed, it may be destroyed with IVEDestroyEventContext.

Cross references
