Reference: IVERegisterCleanupRoutine

Overview

Registers a function that will be called when the process exits. Normally a C or C++ program would use atexit; this routine is intended for use where atexit is not appropriate or there is concern about hitting the atexit's limit on the number of functions registered.

Prototype (C)

#include "ive_processbase.h"
void IVERegisterCleanupRoutine(void (*func)(void*), void* param, int* p_status);

Prototype (Fortran)

There is no Fortran interface.

Parameters

func
func is the routine to register.
param
param is passed at the argument to func when it is called.
p_status
If the call fails, *p_status will be set to one of the values below; otherwise its value is not modified.
IVE_ERR_NO_MEM
There was insufficient memory to store the routine and its argument.
IVE_ERR_NO_RESOURCES
The mechanism to call func when the process exits could not be established.
