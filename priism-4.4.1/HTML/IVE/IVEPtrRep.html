Reference: IVEPtrRep

Overview

Because processes attached to an IVEArena may not may map the shared memory segment to the same location, it is not safe to directly exchange pointer values among the processes. In C, an IVEPtrRep is used to store an encoding of pointer value that may be safely exchanged. Use the function IVEPtrToRep to convert a pointer to an IVEPtrRep and use the function IVEPtrToRep to convert an IVEPtrRep back into a pointer. An IVEPtrRep is guaranteed to be an unsigned integral type.

Include ive_shm.h to declare IVEPtrRep.
