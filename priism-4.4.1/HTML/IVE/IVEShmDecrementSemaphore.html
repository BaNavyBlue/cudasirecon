Reference: IVEShmDecrementSemaphore

Overview

Atomically tests a semaphore, and if its value is greater than zero, decrements the value. Otherwise, the thread joins the set of threads waiting on the semaphore and blocks until it can decrement the semaphore's value.

Prototype (C)

#include "ive_standards.h"
#include "ive_shm.h"
void IVEShmDecrementSemaphore(IVEShmSemaphore* p_sema);

Prototype (Fortran)

There is no Fortran interface.

Parameters

p_sema
p_sema points to the semaphore whose value is to be decremented.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
