Reference: IVEGetPluginAttr

Overview

Returns the address of the plugin's attributes.

Prototype (C)

#include "ive_plugin.h"
void* IVEGetPluginAttr(IVEPlugin plugin);

Prototype (Fortran)

There is no Fortran interface.

Parameters

plugin
(in) Is the handle, returned by IVELoadPlugin, for the plugin whose attributes are to be retrieved.

Return value

The returned value is the address of the plugin's attributes. The plugin controls the format of the attributes; normally they would contain a table of function pointers for the functions that the plugin can perform. A call to unload a plugin invalidates the address to the plugin's attributes.
