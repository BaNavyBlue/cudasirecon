Reference: IVENoticeSignal

Overview

Informs the library that the callback associated with an IVESignalId should be invoked. IVENoticeSignal is the only IVE event processing routine that is safe to call from an asynchronous signal handler; as such, it is a way of integrating asynchronous signals into an event loop. See the documentation for IVEAddSignal for more details.

Prototype (C)

#include "ive_eventloop.h"
void IVENoticeSignal(IVESignalId id, IVEEventContext context);

Prototype (Fortran)

subroutine IVENoticeSignal(id, context)
integer id, context

Parameters

id
id is a value returned by a previous call IVEAddSignal.
context
context is the context to which id belongs.
