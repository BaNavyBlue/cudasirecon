Reference: IVEPermitExecution

Overview

Modifies the permissions of a file to permit execution of that file by the caller.

Prototype (C)

#include "ive_file.h"
void IVEPermitExecution(const char* name, int* p_status);

Prototype (Fortran)

subroutine IVEPermitExecution(name, status)
character*(*) name
integer status

Parameters

name
(in) name is the name of the file to modify. For Fortran callers, any trailing spaces are not considered part of the filename.
status
(out) status (*p_status in C if p_status is not NULL) is set to one of the following if an error occurs:
IVE_ERR_NO_MEM
There was insufficient space for intermediates.
IVE_ERR_NO_PERMISSION
The caller does not have permission to access the file named name or does not have permission to make that file executable.
IVE_ERR_TIMEOUT
Changing the permissions was interrupted by a signal.
IVE_ERR_IO
An I/O error occurred while determining or modifying the file permissions.
IVE_ERR_BAD_FILENAME
name does not exist or is too long.
IVE_ERR_INTERNAL
An error internal to the library was detected.
IVE_ERR_UNRECOGNIZED
An error occurred which the library was not prepared to handle.
