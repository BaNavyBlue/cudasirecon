Reference: IVEThreadCountProcessors

Overview

IVEThreadCountProcessors returns the number of processors currently available in the system.

Prototype (C)

#include "ive_standards.h"
#include "ive_thread.h"
void IVEThreadCountProcessors(void);

Prototype (Fortran)

integer function IVEThreadCountProcessors()

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
