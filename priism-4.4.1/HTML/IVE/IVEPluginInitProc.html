Reference: IVEPluginInitProc

Overview

IVEPluginInitProc is the prototype for the routine used to initialize a plugin when the plugin is loaded with IVELoadPlugin. The arguments and return value for the initialization procedure are described below.

Include ive_plugin.h to declare IVEPluginInitProc.

Prototype (C)

int (*IVEPluginInitProc)(const char* interface_name, void (*error_reporter)(const char* format, ...), void** p_attr, IVEPluginCleanupProc* p_cleanup_func);

Parameters

interface_name
Is the name of the interface requested in the call to IVELoadPlugin.
error_reporter
Is a pointer to a function whose first argument is a printf-style format string; like printf, the remaining arguments depend on the contents of the format string. Use the function pointed to by error_reporter to record the reason for an initialization failure.
p_attr
Set the pointer pointed to by p_attr to the address of the plugin's attributes. IVEGetPluginAttr will return that address to the end user. If you do not set p_attr, IVEGetPluginAttr will return NULL for the attributes' address.
p_cleanup_func
Set the function pointer pointed to by p_cleanup_func to the address of the function that IVEUnloadPlugin will call to free any resources allocated during the plugin initialization. The documentation for IVEPluginCleanupProc describes the expected prototype for the cleanup function.

Return value

If the initialization succeeded, the function should return 0. If the initialization failed, the function should return a non-zero value and use the function pointed to by error_reporter to record a human-readable explanation for the failure.
