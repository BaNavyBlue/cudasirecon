Reference: IVECancelMessageHandler

Overview

Clears the callback triggered by IVEPostMessage for a particular message type. IVECancelMessageHandler(msg, context is equivalent to calling, from C, IVESetMessageHandler(msg, 0, 0, context).

Prototype (C)

#include "ive_eventloop.h"
void IVECancelMessageHandler(IVEMessage msg, IVEEventContext context);

Prototype (Fortran)

subroutine IVECancelMessageHandler(msg, context)
integer msg, context

Parameters

msg
desc is the integer identifying the messages that will no longer trigger callbacks. msg should be between IVE_MESSAGE_MIN and IVE_MESSAGE_MAX; if it is not, the call to IVECancelMessageHandler has no effect.
context
context is the event context to modify.

Cross references

IVEPostMessage, IVESetMessageHandler, IVEEventContext, and IVEMessage
