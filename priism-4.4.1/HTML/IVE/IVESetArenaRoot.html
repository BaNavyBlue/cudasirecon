Reference: IVESetArenaRoot

Overview

IVESetArenaRoot modifies the value stored in the root location of an IVEArena.

Prototype (C)

#include "ive_shm.h"
void IVESetArenaRoot(IVEPtrRep value, IVEArena arena);

Prototype (Fortran)

There is no Fortran interface.

Parameters

value
value is stored in the root location of arena.
arena
arena is the arena on which to operate.

Return value
