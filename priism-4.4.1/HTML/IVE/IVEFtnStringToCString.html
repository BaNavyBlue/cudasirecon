Reference: IVEFtnStringToCString

Overview

Dynamically allocates and initializes a null-terminated character string from a Fortran string.

Prototype (C)

#include "ive_fortran.h"
char* IVEFtnStringToCString(const ftn_char_t* fstr, ftn_length_t flength);

Prototype (Fortran)

There is no Fortran interface.

Parameters

fstr
fstr is the Fortran string to use as a source.
flength
flength is the length of the Fortran string.

Return value

Return 0 if the dynamic allocation failed. Otherwise, a copy of the Fortran string with a null character appended is returned; this copy should be deallocated with free.
