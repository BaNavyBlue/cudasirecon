Reference: IVEAddTimer

Overview

Causes an event to be triggered once a given amount of time has elapsed.

Prototype (C)

#include "ive_eventloop.h"
IVETimerId IVEAddTimer(IVETimeInterval intvl, IVETimerProc proc, void* arg, IVEEventContext context);

Prototype (Fortran)

integer function IVEAddTimer(intvl, proc, arg, context)
integer intvl, context
integer arg(*)
external proc

Parameters

intvl
The event is generated after invtl microseconds have elapsed from the time IVEAddTimer is called. If intvl is negative, it will be treated as if it were zero.
proc
proc is the callback routine. After intvl microseconds have elapsed, a thread which calls IVEProcessEvent or is waiting in an event loop can call proc.
arg
arg is not used internally but is passed along as an argument to proc.
context
context is the context to which the event source will be added.

Return value

Returns a non-zero value if successful. Returns zero if not. When the event source is no longer of interest, it can be removed by passing the returned ID to IVERemoveTimer. The event source is automatically removed after the callback associated with it completes.
