Reference: IVEShmTryAcquireReadLock

Overview

Acquires non-exclusive access to a read-write lock unless it would require blocking until the lock becomes available.

Prototype (C)

#include "ive_standards.h"
#include "ive_shm.h"
int IVEShmTryAcquireReadLock(IVEShmRWLock* p_lock);

Prototype (Fortran)

There is no Fortran interface.

Parameters

p_lock
p_lock points to the lock to be acquired for non-exclusive access.

Return value

If *p_lock could be acquired, the function returns true and the caller should release the lock with IVEShmReleaseReadLock when it no longer needs to hold it. If *p_lock could not be acquired without blocking, the function returns false.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
