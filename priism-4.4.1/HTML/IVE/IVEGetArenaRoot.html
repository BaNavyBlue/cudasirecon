Reference: IVEGetArenaRoot

Overview

IVEGetArenaRoot returns the value stored in a reserved part of an IVEArena. Unrelated processes may use IVEGetArenaRoot and its counterpart, IVESetArenaRoot, to exchange the information needed to start communication.

Prototype (C)

#include "ive_shm.h"
IVEPtrRep IVEGetArenaRoot(IVEArena arena);

Prototype (Fortran)

There is no Fortran interface.

Parameters

arena
arena is the arena on which to operate.

Return value

Returns the value stored in the root location for arena. Until IVESetArenaRoot has been called for the arena, the returned value will be zero.
