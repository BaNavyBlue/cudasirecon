Reference: IVETimerProc

Overview

In C, IVETimerProc is the prototype for callback routines registered with IVEAddTimer. As shown below, it takes three arguments: the argument registered when the callback was registered, the handle assigned to the callback, and the event context. The Fortran interface is similar and is also shown below.

Include ive_eventloop.h to declare IVETimerProc.

Prototype (C)

void (*IVETimerProc)(void* arg, IVETimerId id, IVEEventContext context);

Prototype (Fortran)

subroutine your_name_here(arg, id, context)

integer arg(*)

integer id, context

Parameters

arg
arg is the argument that was registered with IVEAddTimer when the callback routine was registered.
id
id is the ID assigned to the callback when it was registered.
context
context is the event context in which the event occurred.

Cross references

IVEAddTimer, IVERemoveTimer, IVEEventContext, and IVETimerId
