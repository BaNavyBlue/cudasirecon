Reference: IVEProcessEvent

Overview

Processes a single event.

Prototype (C)

#include "ive_eventloop.h"
int IVEProcessEvent(int block, IVEEventContext context);

Prototype (Fortran)

integer function IVEProcessEvent(block, context)
logical block
integer context

Parameters

block
If block is true, IVEProcessEvent will wait until an event occurs. If it is false, IVEProcessEvent will process an event only if there is one currently available.
context
context is the context from which to draw the event.

Side effects

When processing a file descriptor event for a multi-threaded event context, the same type of events on the same file descriptor are temporarily blocked from generating any more calls to callback routines. The documentation for IVEInputProc describes the conditions that end the blockade.

Return value

Returns the number of events processed: either zero or one. With a blocking call to IVEProcessEvent, the returned value will be one except when the context is accessed by multiple threads and one of those threads destroyed the context while another thread was blocked in IVEProcessEvent.
