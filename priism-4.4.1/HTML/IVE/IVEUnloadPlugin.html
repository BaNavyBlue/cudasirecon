Reference: IVEUnloadPlugin

Overview

Unloads a plugin loaded with IVELoadPlugin and frees any resources allocated when the plugin was loaded. When the function returns, the handle to the plugin and any references to the plugin's attributes are no longer valid.

Prototype (C)

#include "ive_plugin.h"
void IVEUnloadPlugin(IVEPlugin plugin);

Prototype (Fortran)

There is no Fortran interface.

Parameters

plugin
(in) Is the handle, returned by IVELoadPlugin, for the plugin to unload.

Cross references
