Reference: IVEShmTryDecrementSemaphore

Overview

Atomically tests a semaphore, and if its value is greater than zero, decrements the value. Otherwise, simply returns without altering the semaphore's value.

Prototype (C)

#include "ive_standards.h"
#include "ive_shm.h"
int IVEShmTryDecrementSemaphore(IVEShmSemaphore* p_sema);

Prototype (Fortran)

There is no Fortran interface.

Parameters

p_sema
p_sema points to the semaphore whose value is to be decremented.

Return value

Returns true if the semaphore's value was decremented; otherwise, returns false.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
