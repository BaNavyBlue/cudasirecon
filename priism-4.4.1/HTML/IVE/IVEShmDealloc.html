Reference: IVEShmDelloc

Overview

Recycles memory allocated with IVEShmAlloc or IVEShmCalloc.

Prototype (C)

#include "ive_shm.h"
void IVEShmDealloc(void* ptr, IVEArena arena);

Prototype (Fortran)

There is no Fortran interface.

Parameters

ptr
ptr is a pointer returned by IVEShmAlloc or IVEShmCalloc.
arena
arena is the arena to which ptr points.

Cross references

IVEShmAlloc, IVEShmCalloc, IVEShmRealloc, and IVEArena
