Reference: IVECreateThreadedEventContext

Overview

Creates the context needed by the event loop functions. A context created by this function may be used from multiple threads. The description of the IVEEventContext type has more details about using an event context with multiple threads.

Prototype (C)

#include "ive_eventloop.h"
IVEEventContext IVECreateThreadedEventContext(void);

Prototype (Fortran)

integer function IVECreateThreadedEventContext()

Return value

Returns a non-zero value if successful. Returns zero if the context could not be created. When the context is no longer needed, it may be destroyed with IVEDestroyEventContext.
