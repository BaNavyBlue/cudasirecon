Reference: IVEShmRealloc

Overview

Changes the size of a block dynamically allocated from an IVEArena.

Prototype (C)

#include "ive_shm.h"
void* IVEShmRealloc(void* ptr, IVEArenaSize sz, IVEArena arena);

Prototype (Fortran)

There is no Fortran interface.

Parameters

ptr
ptr is a pointer returned by a previous call to IVEShmAlloc, IVEShmCalloc, or IVEShmRealloc.
sz
sz is the new size in bytes for the block of memory pointed to by ptr.
arena
arena is the arena on which to operate.

Return value

If unsuccessful, returns zero; otherwise, returns a pointer to a region sufficient for storage of at least sz bytes. The initial minimum(sz, previous size of block) bytes of the block retain their values. The contents of the remainder of the block (if it was extended) are undefined.

When the allocated region is no longer needed, it may be recycled with IVEShmDealloc.

Cross references

IVEShmAlloc, IVEShmCalloc, IVEShmDealloc, IVEArena, and IVEArenaSize.
