Reference: IVERepToPtr

Overview

Converts a result from IVEPtrToRep into a pointer.

Prototype (C)

#include "ive_shm.h"
void* IVERepToPtr(IVEPtrRep rep, IVEArena arena);

Prototype (Fortran)

There is no Fortran interface.

Parameters

rep
rep is result of IVEPtrToRep to be converted.
arena
arena is the shared memory segment which was used for the call to IVEPtrToRep.

Return value

If rep is zero, the result is zero. Otherwise, a non-zero value, is returned. In either case, the result satisfies the following invariant: IVERepToPtr(IVEPtrToRep(ptr, arena), arena) == ptr.
