Reference: IVEShmAcquireWriteLock

Overview

Acquires a read-write lock for exclusive (read/write) access. If another thread has acquired the lock for exclusive access or one or more threads have acquired it for non-exclusive access, the function call will block until the lock can be acquired.

Prototype (C)

#include "ive_standards.h"
#include "ive_shm.h"
void IVEShmAcquireWriteLock(IVEShmRWLock* p_lock);

Prototype (Fortran)

There is no Fortran interface.

Parameters

p_lock
p_lock points to the lock to be acquire for exclusive access.

Side effects

Inclusion of ive_standards.h sets certain feature-set macros. It is safest to include it before any other headers to avoid inconsistent definitions.
