Reference: IVEGetFreeSpace

Overview

IVEGetFreeSpace returns the amount of space currently available on the file system holding a given file.

Prototype (C)

#include "ive_file.h"
int IVEGetFreeSpace(const char* path, int units, int* p_status);

Prototype (Fortran)

integer IVEGetFreeSpace(path, units, status)
character*(*) path
integer units
integer status

Parameters

patch
(in) path is the path to a file or directory on the file system of interest.
units
(in) IVEGetFreeSpace reports amount of available space on the file system as the amount of free space in bytes divided by the value of units with any remainder from the division dropped. units must be greater than zero.
status(out) IVEGetFreeSpace will set status (*p_status in C if p_status is not NULL) to one of the following if an error occurred:
IVE_ERR_NO_MEM
There was insufficient space for intermediates.
IVE_ERR_NO_PERMISSION
The caller does not have permission to search at least one of the directories in path.
IVE_ERR_TIMEOUT
The attempt to get the amount of free space was repeatedly interrupted by a signal.
IVE_ERR_IO
An I/O error occurred while accessing the file system.
IVE_ERR_BAD_FILENAME
path does not exist or is too long.
IVE_ERR_INVALID
units is less than one.
IVE_ERR_OVERFLOW
The amount of free space is too large to be represented by the system library's structures or by the integer return value.
IVE_ERR_INTERNAL
An error internal to the library was detected.
IVE_ERR_UNRECOGNIZED
An error occurred which the library was not prepared to handle.

Return value

On a successful call (status (*p_status in C) is zero before and after the function call), IVEGetFreeSpace returns the number of bytes available on the file system divided by units with any remainder discarded. If an error occurred, IVEGetFreeSpace returns negative one.
