These directories contain the configuration files, help, and executables
for Priism 4.  The libraries and include files needed for IVE programming
are also included.  The documentation is stored in the HTML directory.
Some starting points within that directory are:

PriismRequirementsX86Linux.html  pre-installation requirements for Linux
PriismRequirementsOSX.html       pre-installation requirements for Mac OS X
PriismInstallation.html          installation instructions
RecentChanges.html               release notes
Priism.html                      top-level documentation with links

Information is also available at http://msg.ucsf.edu/IVE/
